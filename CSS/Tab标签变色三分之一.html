# 如何让tab标签只变色三分之一

假设我们有一组简单的tab标签，结构如下：

<div>
    <span class="tab active">Tab 1</span>
    <span class="tab">Tab 2</span>
    <span class="tab">Tab 3</span>
</div>

.tab {
    display: inline-block;
    padding: 10px 20px;
    border: 1px solid #ccc;
    cursor: pointer;
}

.tab.active {
    background-color: #007BFF;
    color: white;

# 1、使用伪元素和绝对定位

首先修改 CSS 中 .tab.active 的样式定义：

.tab.active {
    position: relative;
    color: white;
    /* 清除之前设置的全背景色 */
    background-color: transparent;
}

.tab.active::before {
    content: "";
    position: absolute;
    left: 0;
    top: 0;
    width: 33.33%; /* 设置宽度为标签宽度的三分之一 */
    height: 100%;
    background-color: #007BFF;
    z-index: -1; /* 将伪元素置于标签元素下方 */
}

通过给激活状态的 tab 标签添加一个伪元素 ::before，并利用绝对定位将其放置在标签的左上角，设置其宽度为标签宽度的三分之一，从而实现只变色三分之一的效果。

# 2、使用 CSS 渐变（linear gradient）

同样先修改 .tab.active 的样式定义：

.tab.active {
    background: linear-gradient(to right, #007BFF 33.33%, transparent 33.33%);
    color: white;
}

利用 CSS 的线性渐变属性，设置从左到右的渐变，在三分之一处由指定颜色（这里是 #007BFF）渐变到透明，这样就使得标签看起来只变色了三分之一。

# 3、使用 JavaScript

使用 JavaScript 动态修改样式（如果需要根据用户交互等动态实现） 在 HTML 文件中添加以下 JavaScript 代码：

<script>
    // 获取所有的tab标签元素
    const tabs = document.querySelectorAll('.tab');

    // 遍历每个tab标签元素
    tabs.forEach(tab => {
        tab.addEventListener('click', function () {
            // 先移除所有tab标签的激活状态样式
            tabs.forEach(t => t.classList.remove('active'));

            // 给当前点击的tab标签添加激活状态样式
            this.classList.add('active');

            // 获取当前激活的tab标签宽度
            const activeTabWidth = this.offsetWidth;

            // 使用JavaScript动态修改当前激活tab标签的背景色，只变色三分之一
            this.style.background = `linear-gradient(to right, #007BFF ${activeTabWidth * 0.3333}px, transparent ${activeTabWidth * 0.3333}px)`;
        });
    });
</script>

通过 JavaScript 监听每个 tab 标签的点击事件，当点击一个 tab 标签时，先移除所有标签的激活状态，然后给当前点击的标签添加激活状态，并根据当前激活标签的实际宽度，动态设置其背景色为线性渐变，实现只变色三分之一的效果。

这种方法在需要根据用户交互动态改变 tab 标签样式时比较有用。

方法一和二相对简单直接，适合静态页面的设置；方法三则更适合需要根据用户交互等动态改变 tab 标签样式的场景。
