# TS 核心

# 1、类型推断

let str = 'abc'
str = 10
// Type 'number' is not assignable to type 'string'.
1
2
3

typescript 会根据你给变量存放的初始值，来进行变量类型限定。

你刚开始给变量的是字符串，这个变量就只能存放字符串类型的值。

typescript根据已书写的代码，进行类型推断的方式。

# 2、类型注解

在书写变量的时候，可以给变量一个进行一个类型的限定。

通过冒号形式来书写

let str:string = 'abc'
// 也可以先进行一个预声明
let str:string
str = 'abc'

# 7、枚举

枚举需要使用一个新的关键字 enum, 后面书写一个需要声明的枚举类型名称。

enum MyEnum {
    A,
    B,
    C
}

// 两种访问方式
console.log(MyEnum.A)
console.log(MyEnum[0])

var MyEnum;
(function (MyEnum) {
    MyEnum[MyEnum["A"] = 0] = "A";
    MyEnum[MyEnum["B"] = 1] = "B";
    MyEnum[MyEnum["C"] = 2] = "C";
})(MyEnum || (MyEnum = {}));
// 两种访问方式
console.log(MyEnum.A);
console.log(MyEnum[0]);

# 8、void

void表示空，类似于null 和 undefined，在使用的时候，void只能被分配undefined值。

# 9、函数

function foo(a, b) {
    return a + b
}
// Parameter 'a' implicitly has an 'any' type.

// 给参数分配一个数据类型
function foo(a: number, b: string) {
    return a + b
}
// 给返回值做分配
// void表示没有返回值
function foo(a: number, b: string): void {
    // ...
}

// 可选参数，可选参数应该在所有参数的后面
function foo(a: number, b?: number): number {
    return a + b
}

// 剩余参数,rest是一个数组结构，需要指定数组类型
function foo(a: number, b?: number, ...rest: string[]): number {
    return a + b
}

const f = foo(20, 30, 'a')

# 10、接口

关键字 interface ,通常定义一个对象

interface Obj {
    name: string,
    age: number
}

const obj: Obj {
    name: 'a',
    age: 10
}

// 避免出现数据类型和键的书写错误
const obj: Obj {
    name: 'a',
    ago: 10
}


interface Obj {
    name: string,
    age: number
}

const obj: Obj = {
    name: 'a',
    ago: 10
}

// Object literal may only specify known properties, and 'ago' does not exist in type 'Obj'.

# 11、type

如果需要同时使用多种类型，可以自定一个类型别名

type MyType = string | number
let a: MyType = 'abc'
// 或者
let a: MyType = 10
// 都是可以的

# 12、泛型

function foo<T>(a: T, b: T): T[] {
    return [a, b]
}

// 以下写法都是可以的，可以不写<>，TS会自动进行类型推断
foo<number>(1, 2)
foo(1, 2)

foo<string>('a', 'b')
foo('a', 'b')
