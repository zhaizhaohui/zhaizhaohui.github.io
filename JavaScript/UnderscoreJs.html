# UnderscoreJs

Underscor.js定义了一个下划线对象，函数库的所有方法都属于这个对象。 这些方法大致上可以分成： 集合(collection) 数组(array) 函数(function) 对象(object) 工具(utility)

# 1、数组处理

map方法：对集合的每个成员依次进行某种操作，将返回的值依次存入一个新的数组。

_.map([1, 2, 3], function(num){ return num * 3; });
_.map({one : 1, two : 2, three : 3}, function(num, key){ return num * 3; });
1
2

each方法：与map类似，依次对数组所有元素进行某种操作，不返回任何值。

_.each([1, 2, 3], alert);
_.each({one : 1, two : 2, three : 3}, alert);
1
2

reduce方法：依次对集合的每个成员进行某种操作，然后将操作结果累计在某一个初始值之上，全部操作结束之后，返回累计的值。 该方法接受三个参数：被处理的集合，对每个成员进行操作的函数，累计用的变量。

_.reduce([1, 2, 3], function(memo, num){ return memo + num; }, 0);
1

shuffle方法：返回一个打乱次序的集合。

_.shuffle([1, 2, 3, 4, 5, 6]);
1

# 2、数组特征

Underscore.js提供了一系列方法，判断数组元素的特征，这些方法都返回一个布尔值，表示是否满足条件。

every方法：判断数组的所有元素是否都满足某个条件。如果都满足则返回true，否则返回false。(类似于逻辑与)

some方法：只要有一个元素满足，就返回true，否则返回false。(类似于逻辑或)

size方法：返回集合的成员数量

# 3、数组过滤

Underscore.js提供了一系列方法，用于过滤数组，找到符合要求的成员。

filter方法：依次对集合的每个成员进行某种操作，只返回操作结果为true的成员。

reject方法：只返回操作结果为false的成员。

find方法：依次对集合的每个成员进行某种操作，返回第一个操作结果为true的成员。如果所有成员的操作结果都为false，则返回undefined。

contains方法：表示如果某个值在数组内，则返回true，否则返回false。

countBy方法：依次对集合的每个成员进行某种操作，将操作结果相同的成员算作一类，最后返回一个对象，表明每种操作结果对应的成员数量。

# 4、对象相关方法

toArray方法：将对象转为数组，只包含对象成员的值。典型应用是将对类似数组的对象转为真正的数组。

pluck方法：将多个对象的某一个属性的值，提取成一个数组。
