# textarea高度自适应

# 需求

随文字增多，将textarea自动撑开

# 解决思路

使用pre标签解决，因为pre标签能保持文本的格式 将textarea的文本内容和样式严格复制一份给pre标签，pre的高度值就为我们所需的textarea的高度值。 然后将pre隐藏，把pre的高度赋值给textarea

# HTML代码

<div>
    <pre id="pre"></pre>
    <textarea id="content"></textarea>
</div>

# CSS样式代码

pre,textarea {
    width: 100%;
    /*这里的宽度可以根据实际情况设置*/
    padding: 15px;
    overflow: hidden
    line-height: 24px;
    min-height: 120px;
    font-family: '微软雅黑', Helvetica, sans-serif;
    word-wrap: break-word;
    /*设置换行，因为pre标签默认是不换行的*/
    /*其他样式可以根据实际情况添加*/
}
pre {
    /*这里不能用display:none，因为这样设置是没有高度的*/
    visibility: hidden;
    position: absolute;
    z-index: -100;
    word-wrap: break-word;
}

# JS代码

document.onkeydown = function () {
    var textarea = document.getElementById('content')
    var pre = document.getElementById('pre');
    pre.innerHTML = textarea.value;
    var realHeight = pre.offsetHeight;
    //offsetHeight = height + padding + border
    if(realHeight > 120) {
        textarea.style.height = realHeight + 'px';
    } else {
        textarea.style.height = realHeight;
   }
}
