# Js数组操作

案例数组

var a = [1,2,3,4,5]
1

1、删除原数组第一项

a.shift()
// [2,3,4,5]
1
2

2、删除原数组最后一项

a.pop()
// [1,2,3,4]
1
2

3、添加到数组开头，并返回数组长度

a.unshift(-1, -2)
// [-1,-2,1,2,3,4,5]
1
2

4、添加到数组尾部，并返回数组长度

a.push(6)
// [1,2,3,4,5,6]
1
2

5、添加到数组尾部，并返回一个新数组

a.contact(6)
// [1,2,3,4,5,6]
1
2

6、数组反序

a.reverse()
// [5,4,3,2,1]
1
2

7、按指定的参数对数组进行排序

a.sort()
// [1,2,3,4,5]
1
2

8、返回从原数组中指定开始下标到结束下标之间的项组成的新数组

a.slice(2,5)
// [3,4,5]
1
2

9、将数组的元素组起一个字符串，以separator为分隔符，省略的话则用默认用逗号为分隔符

a.join('|')
// '1|2|3|4|5'
1
2

10、从指定下标位置删除指定长度的项，并插入新的内容。返回删除后的数组 一般用于数组的替换和指定项的删除

var b = a.splice(2,2,7,8,9)
// a:[1,2,7,8,9,5]
// b:[3,4]
1
2
