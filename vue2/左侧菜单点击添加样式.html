# 左侧菜单点击添加样式

# 场景

后台管理系统，左侧菜单。菜单有二级菜单，在点击二级菜单的时候，该条目添加class样式，其他条目删除class样式。右侧的当前地址栏显示该条目的名称。

# 数据结构

    {
        tag: '用户管理',
        childs: [
            {
                name: '系统用户',
                url: '/sysUser'
            },
            {
                name: '管理员',
                url: '/sysManger'
            }
        ]
    },
    {
        tage: '内容管理',
        childs: [
            {
                name: '新闻管理',
                url: '/news'
            },
            {
                name: '图片管理',
                url: '/photos'
            }
        ]
    }
]

# 解决方案

在循环二维数据的时候，为循环条目添加动态样式。其中k为一维数组的下标，key为二维数组的小标。 <div v-text="val.title" 

:class="{'cur': (k + '-' + key)==isCur}"
@click="changeTabName(k, key)"
></div>

如果数据结构是一维数组，只需要用到k就可以。即：

<div
v-text="val.title"
<!--重点在于此处-->
:class="{'cur': k==isCur}"
@click="changeTabName(k)"
></div>

# 方法说明

由于在点击条目的同时，需要动态添加class样式和改变右侧地址栏名称。所以没有使用router-linke跳转，而是使用了changeTabName方法。

changeTabName (k, i) {
	let t = this.sideList[k]
	let name = t.tag + ' / ' + t.childs[i].title
	this.cardName = name
	// 添加样式
	let cur = k + '-' + i
	this.isCur = cur
	// 使用vuex保存状态，在页面刷新的时候，class样式和栏目名称不会丢失
    this.$store.commit('cardOpts', {isCur: cur, cardName: name})
    // 跳转路由
	this.$router.push(t.childs[i].url)
}
