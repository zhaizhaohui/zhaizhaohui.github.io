# 脚手架的搭建

1、从node官网下载并安装node。

2、安装cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org
1

3、安装vue和vue-cli

cnpm install vue / npm install vue
cnpm install --global vue-cli
1
2

4、初始化项目

vue init webpack my-project
cd my-project
1
2

5、安装项目依赖

npm install
1

6、启动项目

npm run dev
1

7、访问项目

http://localhost:8080
