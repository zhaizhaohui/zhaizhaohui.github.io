# 父子组件之间的传值

# 父组件向子组件传值

# 1、在父组件中通过属性的形式向子组件传值

<child-components :city="citys"></child-components>
1

# 2、在父组件中的js中

export default {
	...
	data () {
		return {
			// 此处citys是为了与传过去的city区分，更加明了数据传递的命名
			// 当然在传递的时候也可以命名为city
			citys: ['a',...]
		}
	}
}

# 3、props

在子组件中的js中，使用props接受传过来的数据

export default {
	...
	props: {
		// prop是以对象的形式书写的
		city: {
			// type形式也可以是字符串或其他
			type: Array,
			required: true
		}
	}
}

# 子组件向父组件传值

# 1、在子组件中定义一个传值的方法

<butto @click="childClick">点我</button>
1

# 2、$emit

在子组件中的js中通过$emit向父组件传值

methods: {
	......
  	childClick () {
  		// 第一个参数childByValue是在父组件on监听的方法
		// 第二个参数this.childValue是需要传的值
    	this.$emit('childByValue', this.childValue)
  	}
}

# 3、on

在父组件中引入子组件，定义一个监听子组件状态的方法(on)

<child v-on:childByValue="childByValue"></child>
1

# 4、childValue

在父组件的js中，通过childValue接受子组件传过来的值

export default {
	components: {
		child
	},
	 methods: {
			childByValue (childValue) {
			this.name = childValue
	  	}
	}
}
