# Vuex

# 1、Vuex概念

Vuex 是一个专为 Vue.js 应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

# 2、理念流程：

a、Components调用Actions b、Actions提交到Mutations c、Mutations改变State d、State渲染到Components

# 3、安装vue

npm install vuex --save

# 9、actions和mutations

在store目录下的index.js中创建actions和mutations

actions: {
	// city为vue中传过来的参数
	changeCity (ctx, city) {
		// 这里的changeCity是mutations中的changeCity
		ctx.commit('changeCity', city)
	}
},
mutations: {
	changeCity (state, city){
		state.city = city
	}
}

# 10、commit

如果组件改变state的过程中没有任何异步操作，则可以直接在组件中使用commit直接调用mutations，而省略掉actions vue组件中

this.$store.commit('changCity',city)

# 13、vuex的映射

在vue组件中

import { mapState } from 'vuex'
export default {
	...
	computed: {
		// mapState是把vuex中的数据映射到computed计算属性中city计算属性
		...mapState(['city'])
	}
}

# 14、mutations映射

在methods中使用mutations映射

import { mapState, mapMutations } from 'vuex'
methods: {
	// 把名字为changeCity的mutation映射到组件中叫做changeCity的方法里
	...mapMutations(['changeCity'])
}
