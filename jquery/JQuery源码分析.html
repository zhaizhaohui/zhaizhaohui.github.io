# 23、【21~94】代码分析

# 为什么传参window

1、window是js最顶端，查找较慢。传参后找的是参数(局部变量)，而不是外边的window 2、便于压缩

# 为什么要传undefined

undefined是window下的一个属性，不是一个保留字也不是关键字，undefined在外边是可以被修改的，传参undefined是为了防止undefined被修改

# "use strict"严格模式

jquery中并不推荐使用，原因是兼容性，asp.net的引起火狐的假死，Support: Firefox 18+

# 定义rootjQuery

1、便于压缩 2、便于后期代码的维护
